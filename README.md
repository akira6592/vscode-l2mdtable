# Convert list to Markdown table

## Features
### Convert list to Markdown table
![Screen Shot](/images/ss01_convert.gif)

#### before
```markdown
- 
  - Header 1
  - Header 2
  - Header 3
- 
  - Column 1-1
  - Column 1-2
  - Column 1-3
- 
  - Column 2-1
  - Column 2-2
  - Column 2-3

```
#### after 
```markdown
|Header 1|Header 2|Header 3|
|:------|:------|:------|
|Column 1-1|Column 1-2|Column 1-3|
|Column 2-1|Column 2-2|Column 2-3|
```
